Jboss Outreach App
===================================

This app displays a list of repositories of JBossOutreach organisation on github

Pre-requisites
--------------

- Android SDK v23
- Android Build Tools v23.0.2
- Android Support Repository v23.3.0

Libraries
-------------
-Picasso
